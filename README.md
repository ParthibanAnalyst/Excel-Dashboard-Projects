# Excel Dashboard Projects

## Projects Overview
Interactive dashboards built using Microsoft Excel analyzing real-world data.

## Projects

### 1. Hospital Dashboard
- Analyzed hospital patient data
- City-wise, department-wise analysis
- Tools: Excel, Pivot Tables, Power Query, Slicers

### 2. Sales Dashboard
- Regional sales analysis
- Total Revenue: 94,768,000
- Tools: Excel, Pivot Tables, Charts

### 3. Student Dashboard
- Department-wise average analysis
- Gender-wise grade comparison
- Tools: Excel, Pivot Tables

## Skills Used
SQL | Excel | Power BI | Power Query | Slicers
